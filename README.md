## Налаштування середовища

1. Створіть віртуальне середовище:
   ```bash
   python -m venv venv
   ```

2. Активуйте віртуальне середовище:

   На Windows:

   ```bash
   venv\Scripts\activate
   ```
   На macOS/Linux:

   ```bash
   source venv/bin/activate
   ```

3. Встановіть залежності:

   ```bash
   pip install -r requirements.txt
   ```
